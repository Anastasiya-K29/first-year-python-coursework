# Year 1 Python

Python assignments and labs completed during Year 1 of BSc Computer Science 
at Dublin City University (2024–2025).

## Topics Covered

- Core Python: strings, lists, tuples, dictionaries, sets
- File I/O and exception handling
- List comprehensions and functional tools
- Object-oriented programming: classes, special methods, operator overloading
- Data structures: stacks and queues
- Recursion
- Graph theory: depth-first and breadth-first search
- Regular expressions

## Structure

Each folder corresponds to a topic area and contains lab exercises and assignments.

## Language
Python 3
